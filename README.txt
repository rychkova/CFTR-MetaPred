# CFTR-MetaPred

This repository includes data from the following paper:

"Developing gene-specific meta-predictor of variant pathogenicity", A. Rychkova, M. C. Buu, C. Scharfe, M. I. Lefterova, J. I. Odegaard, I. Schrijver, C. Milla, C. D. Bustamante

Two files are included:

1) all_1899_CFTR_vars.xlsx - complete list of 1,899 protein-coding CFTR variants with clinical significance, allele frequency and corresponding source dataset name;

2) missense_1210_CFTR_vars.xlsx - 35 annotation features for 1,210 missense variants together with RF predictions.
